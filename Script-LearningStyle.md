## 📝 เวอร์ชัน A — TikTok Script (60–90 วินาที)

⸻

🪝 **HOOK (0–3 วิ)**

ลูกคุณนั่งเรียนไม่ได้
วอกแวก ขยับตัวตลอด
ครูบอกว่า "สมาธิสั้น"

แต่รู้ไหม...
บางทีมันไม่ใช่ปัญหาของลูก
**มันคือ Mismatch ระหว่าง Learning Style กับห้องเรียน**

⸻

😩 **PAIN POINT (3–15 วิ)**

พ่อแม่หลายบ้านกังวล
เพราะลูกนั่งเฉยไม่ได้แม้แต่ 10 นาที
บางคนพาไปหาหมอวินิจฉัย ADHD

แต่งานวิจัยพบว่า
เด็กกลุ่มหนึ่งต้องการ **Movement** เพื่อให้สมองทำงาน
ไม่ใช่ทุกคนที่ขยับตัวเยอะ = มีปัญหา

⸻

🔄 **REFRAME (15–25 วิ)**

ลูกทุกคนมี **Learning Style** ที่ต่างกัน
และนั่นไม่ใช่ความผิดปกติ
**มันคือการที่สมองถูก wire มาต่างกัน**

⸻

🧠 **SCIENCE — 4 Learning Styles (25–55 วิ)**

1️⃣ **Visual Learner — เรียนด้วยการดู**
 • จำได้เมื่อเห็นภาพ แผนผัง สี
 • อ่านหนังสือ ดูรูป วาด mindmap
 • ห้องเรียนที่ดีสำหรับเขา: มีบอร์ด มีสี มีรูปภาพ

2️⃣ **Auditory Learner — เรียนด้วยการฟัง**
 • จำได้เมื่อได้ยิน ได้พูด ได้อธิบาย
 • ชอบฟังครูบรรยาย ท่อง คุย discuss
 • ห้องเรียนที่ดีสำหรับเขา: มีเสียง มี discussion

3️⃣ **Read/Write Learner — เรียนด้วยการอ่าน-เขียน**
 • จำได้เมื่อได้จด ได้เขียน ได้อ่าน
 • ชอบ flashcard โน้ต การบ้านเขียน
 • ห้องเรียนที่ดีสำหรับเขา: มีหนังสือ มีการเขียน

4️⃣ **Kinesthetic Learner — เรียนด้วยการลงมือทำ**
 • จำได้เมื่อได้ทำ ได้แตะ ได้เคลื่อนไหว
 • **นี่คือเด็กที่ "ซนที่สุด" ในสายตาครูทั่วไป**
 • ห้องเรียนที่ดีสำหรับเขา: Project-based, Lab, กิจกรรม

⸻

📊 **RESEARCH (55–70 วิ)**

งานวิจัยจาก Wiley Interdisciplinary Reviews (2026) พบว่า
เด็กที่ได้ **Movement** อย่างเพียงพอ
มี attention, memory และ Executive Function ที่ดีขึ้น

และงานวิจัยจาก Children (2025) พบว่า
เด็กที่ได้ออกกำลังกาย**ตอนเช้า**ก่อนเรียน
มีคะแนน Attention สูงขึ้นอย่างมีนัยสำคัญ

พูดง่ายๆ คือ **Kinesthetic Learner ≠ เด็กมีปัญหา**
แค่ต้องการสภาพแวดล้อมที่ต่างออกไป

⸻

⚠️ **CONDITION (70–80 วิ)**

แต่...
Learning Style ≠ ข้ออ้างสำหรับทุกอย่าง

ถ้าลูกวอกแวกมากจนกระทบชีวิตประจำวัน
 → ควรปรึกษาจิตแพทย์หรือนักจิตวิทยาเด็ก

Kinesthetic Learner ≠ ADHD
แต่ทั้ง 2 อย่างก็มีได้พร้อมกัน

⸻

**แต่คือโรงเรียนที่รู้ว่า
ลูกสมองถูก wire มาแบบไหน
แล้วสอนให้ตรงนั้น**

## 📝 เวอร์ชัน B — Neuroscience Deep-dive (Carousel / Long-form)

⸻

🪝 **HOOK + METAPHOR**

ลองนึกภาพ...
ลูกคุณถูกบังคับให้เขียนด้วยมือซ้าย
ทุกวัน 8 ชั่วโมง ตลอด 12 ปี

นั่นแหละที่เกิดขึ้น
เมื่อ Kinesthetic Learner ถูกส่งไปโรงเรียนที่บังคับให้นั่งนิ่งตลอดเวลา

⸻

🧠 **PART 1 — Sensory Processing Network**

สมองเด็กแต่ละคนมี **Sensory Processing Network** ที่ dominant ต่างกัน

 • บางคน → **Visual Cortex (V1–V4)** active มากกว่า
   → เรียนรู้ได้ดีที่สุดเมื่อเห็นภาพ

 • บางคน → **Auditory Cortex + Wernicke's Area** active มากกว่า
   → เรียนรู้ได้ดีที่สุดเมื่อได้ยินและพูด

 • บางคน → **Motor Cortex + Cerebellum** active มากกว่า
   → เรียนรู้ได้ดีที่สุดเมื่อได้เคลื่อนไหว

นี่ไม่ใช่ความผิดปกติ
**มันคือ Neural Preference ที่ถูก shaped โดย genetics + early experience**

⸻

🧠 **PART 2 — Dopamine + Movement Loop**

Kinesthetic Learner มีระบบ **Dopaminergic Reward Circuit** ที่ตอบสนองต่อ Movement สูงกว่า

เมื่อเขาขยับตัว:
 • Dopamine หลั่ง → สมองตื่นตัว → Attention เพิ่ม
 • Hippocampus ทำงานดีขึ้น → Memory Encoding แม่นขึ้น
 • Prefrontal Cortex (PFC) ได้รับ blood flow มากขึ้น → Executive Function ดีขึ้น

แต่เมื่อเขาถูกบังคับให้นั่งนิ่ง:
 • Dopamine ต่ำ → สมองหาวิธี self-stimulate เอง
 → ผลคือ วอกแวก ขยับตัว ดูเหมือนไม่ตั้งใจ

งานวิจัย Leisman et al. (2026) — Wiley Interdiscip Rev Cogn Sci
DOI: [10.1002/wcs.70020](https://doi.org/10.1002/wcs.70020)
พบว่า Movement สัมพันธ์กับ Attention, Memory และ Executive Function
ในเด็กตั้งแต่วัยเรียนจนถึงวัยรุ่น

⸻

🧠 **PART 3 — The Timing Effect (เวลาสำคัญกว่าที่คิด)**

งานวิจัย Latino et al. (2025) — Children (Basel)
DOI: [10.3390/children12101324](https://doi.org/10.3390/children12101324)

ทดลองกับเด็กอายุ 12–13 ปี 102 คน
แบ่งออกกำลังกาย 3 ช่วง: เช้า / กลางวัน / บ่าย

ผลลัพธ์:
 • **ออกกำลังกายตอนเช้า** → Attention (η²p = 0.16) สูงสุด
 • Working Memory ดีขึ้นชัดเจนที่สุด
 • **Alertness สูงกว่าช่วงอื่นอย่างมีนัยสำคัญ (η²p = 0.19)**

สิ่งที่นี่แปลว่า:
Kinesthetic Learner ควรได้ขยับก่อนเรียน
ไม่ใช่ถูกบังคับนั่งนิ่งก่อน
แล้วถึงจะ "ตั้งใจเรียนได้"

⸻

🧠 **PART 4 — Sensory Input แตกต่างกัน = ผลการเรียนต่างกัน**

งานวิจัย Söderlund et al. (2021) — Brain and Behavior
DOI: [10.1002/brb3.2114](https://doi.org/10.1002/brb3.2114)

พบว่าเด็กที่มี **Reading Disability (Dyslexia)**
เรียนได้ดีขึ้นเมื่อได้รับ **White Noise** ขณะอ่าน
(ผ่านกลไก Stochastic Resonance)

ขณะที่เด็กทั่วไปกลับ **แย่ลง** เมื่อมี White Noise

นั่นแสดงว่า Sensory Input ที่ "ดี" สำหรับคนหนึ่ง
อาจ "แย่" สำหรับอีกคน

**→ ไม่มีห้องเรียนที่สมบูรณ์แบบสำหรับทุกคน
มีแต่ห้องเรียนที่ match หรือ mismatch กับ sensory profile**

⸻

🧠 **PART 5 — มุมคิดเพิ่ม**

คำถามที่ลึกกว่าคือ...

เราออกแบบโรงเรียนเพื่อ **ใคร**?

โรงเรียนส่วนใหญ่ถูกออกแบบมาสำหรับ
Read/Write Learner + Auditory Learner

เพราะว่า ประวัติศาสตร์การศึกษาเกิดขึ้นในยุคที่
การอ่าน-เขียน-ฟัง คือทักษะที่ valued ที่สุด

แต่โลกปัจจุบัน valued:
 • Problem-solving (ต้องลงมือทำ = Kinesthetic)
 • Collaboration (ต้องพูดคุย = Auditory)
 • Visual thinking (ต้องนึกภาพ = Visual)

**คำถามคือ ลูกของคุณถูก wire มาเพื่อโลกอนาคต
แต่โรงเรียนยังออกแบบมาสำหรับโลก 100 ปีที่แล้ว**

⸻

🎯 **สรุปแบบหมอกวาง**

ลูกที่นั่งนิ่งไม่ได้
ไม่ได้แปลว่าลูกไม่ฉลาด
ไม่ได้แปลว่าลูกสมาธิสั้น

**มันอาจแปลว่า
สมองของเขาต้องการ Movement เพื่อ encode memory**

และโรงเรียนที่ดีที่สุด
คือโรงเรียนที่ไม่บังคับให้ลูกเป็นคนที่เขาไม่ได้ถูก wire มาเป็น
